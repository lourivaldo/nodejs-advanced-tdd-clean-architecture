# Nodejs Advanced TDD Clean Architecture

Repo: https://github.com/rmanguinho/advanced-node

## Requirements
- NodeJS v14.17.4
- Docker 20.10.14
- Docker Compose 1.25.420

## Installing
```
npm install
npm run build
```

## Running
```
docker-compose up --build
npm start
```

## Tests
```
npm test
npm test:fb-api
```
